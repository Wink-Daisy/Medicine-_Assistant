# 💊 吃药小管家 - 老年人用药提醒系统

## 📖 项目简介

《吃药小管家》是一款专为老年人设计的用药提醒软件。针对老年人记忆力衰退、视力下降等问题，提供药品管理、定时提醒、语音播报、服药记录等功能，帮助老年人按时、按量服用药物。

## ✨ 主要功能

| 功能模块 | 功能说明 |
|----------|----------|
| 📋 药品管理 | 添加、编辑、删除药品，支持上传药品照片 |
| ⏰ 定时提醒 | 到点自动弹出全屏提醒窗口 |
| 🔊 语音播报 | 自动朗读药品名称和剂量，循环播报1分钟 |
| ✅ 服药记录 | 标记已服用，记录服药时间 |
| 📅 日历视图 | 按月查看服药记录，直观显示完成情况 |
| ⚙️ 系统设置 | 紧急联系人、超时时间、数据备份恢复 |

## 🖥️ 技术栈

| 技术 | 说明 |
|------|------|
| Python 3.9 | 开发语言 |
| Tkinter | GUI界面框架 |
| pyttsx3 / win32com | 语音播报引擎 |
| Pillow | 图片处理 |
| JSON | 数据存储 |
